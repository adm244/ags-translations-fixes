# AGS Translation fixes

This repo contains a collection of game specific script fixes for AGS games.

The goal is to make each AGS game fully translatable and fix translation related issues.

Keep in mind that some fixes may be out-of-date. Feel free to report any issues.

## Tools

These tools are used to create patches:

- [AGSUnpacker](https://github.com/adm244/AGSUnpacker) by AdM244
- [Ghidra-ReAGS](https://github.com/adm244/Ghidra-ReAGS) by AdM244
- [agsutils](https://github.com/rofl0r/agsutils) by rofl0r
- [ascc](https://github.com/rofl0r/ascc) by rofl0r
